Display and Positioning

Properties regarding the HTML elements' position in the browser

Property Description Values Effect
position Specifies the type of positioning method used for an element. Top, bottom, left and right can be specified. Admits px, em and rem values. static Default element positioning
relative Position to its default static position
absolute Position to its closest positioned parent element. Removes the element from the html flow.
fixed Position fixed to a specific position anywhere in the webpage. Removes the element from the html flow.
sticky Sticks to a specific position as the page is scrolled.
display Specifies how an element is shown on a web page. Width and height properties can be specified. inline Default element positioning. Makes any element an inline html.
block Displays the element alone, taking all horizontal space.
inline-block Position elements inline next to other.
z-index Specifies the stack order of an element An integer An element can have a positive or negative stack order (1, -3, etc.)
float Specifies how an element should float. left The element floats to the left of its container.
right The element floats to the right of its container.
none The element does not float (will be displayed just where it occurs in the text). This is default.
inherit The element inherits the float value of its parent.
clear Specifies what should happen with the element that is next to a floating element. left The element is pushed below left floated elements.
right The element is pushed below right floated elements.
both The element is pushed below both left and right floated elements.
none The element is not pushed below left or right floated elements. This is default.
inherit The element inherits the clear value from its parent.
